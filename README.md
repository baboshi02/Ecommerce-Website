# Ecommerce Website
Simple Ecommerce website that uses fake data. Implemented with React js.

## Prerequisites
-Node JS

## Installation

```bash
git clone https://github.com/baboshi02/Ecommerce.git

cd Ecommerce

npm install

npm run dev
```
